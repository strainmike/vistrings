# vistrings

Library for extracting plaintext from LabVIEW VIs.  This is intended for ease of searching VIs.

# Usage

    pip install vistrings
    python -m vistrings <file>

# Example

    vistrings tests\strings.vi
    Pane
    Label Contents
    Comment Contents
    String Constant Contents
